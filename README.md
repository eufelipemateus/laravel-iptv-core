# Laravel IPTV Core - FelipeMateus\IPTVCore

## Instaling

Tested in Laravel 9.11.

### Run the command below in root to install the package in your project

```bash

composer require felipemateus/iptv-core  

```

### Migrate the database
  
```bash
php artisan migrate
```
